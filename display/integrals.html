Integrals

  1. (Stewart) Evaluate \(\int \frac{\mathrm{d}x}{x^7-x}\). The straightforward approach would be to start with partial fractions, but that would be brutal. Try a substitution.
  2. (Stewart) Evaluate \(\displaystyle\int_0^1 \sqrt[3]{1-x^7} - \sqrt[7]{1-x^3} \,\mathrm{d}x\,.\)
  3. (Stewart) If \(n\) is a positive integer, prove that \[\int\limits_0^1 \big(\ln(x)\big)^n \,\mathrm{d}x = (-1)^n n!\,.\]
  4. (Ubiquitous) Evaluate the following integrals.
    1. \(\displaystyle\int \frac{1}{x^5+1} \,\mathrm{d}x\)
    2. \(\displaystyle\int \sqrt{\tan(x)} \,\mathrm{d}x\)
